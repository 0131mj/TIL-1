# 파이참 단축키
- cmd+O : find class or file by name
- cmd g: 동일 단어 하나씩 선택
- 원하는 선택-> cmd g: 동일 단어 하나씩 더 선택 (되돌리는건 shift)
- cmd f -> option enter: 찾은 단어 전체 선택
- opt+방향키: 선택 범위 넓히기/좁히기
- ctrl+space: 코드 컴플리션
- cmd shift enter: complete statement
- cmd shift delete: 바로 전 커서위치로 가기
